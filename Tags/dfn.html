Tag usada para definir a instância de definição de um termo. A instância de definição normalmente é primeiro uso de um termo. A tag pai de dnf também deve conter a explicação para o termo.

HTML é uma linguagem de marcação de hypertext.

OBS: Note que, como a palavra 'HTML' foi escrita pela primeira vez, colocamos uma tag dfn, pois logo em seguida, no próprio parágrafo, há-se a explicação do que seria o termo mencionado.
